# Afterfall Client 1.0.4

Patch client focalizzata sulle icone buff UI.

Questa release pubblica:

- `afterfall_patch_1.0.1_to_1.0.4.zip`
- `afterfall_patch_1.0.2_to_1.0.4.zip`
- `afterfall_patch_1.0.3_to_1.0.4.zip`

Contenuto funzionale:

- icona + timer per le Acque Biologo
- icona + timer per il buff Companion Spirituale
- Companion aggiornato da `gold_premium.sub` a `support/summon.sub`
- nessun cambio gameplay, server o asset grafici

La patch `1.0.1 -> 1.0.4` include anche:

- `Metin2_RelWithDebInfo.exe` pubblico gia' validato nella `1.0.2`

Le patch `1.0.2 -> 1.0.4` e `1.0.3 -> 1.0.4` aggiornano solo:

- `pack/root.pck`
- `version.txt`

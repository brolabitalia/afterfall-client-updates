# Afterfall Client 1.0.5

Patch client focalizzata su descrizioni consumabili e hint item.

Questa release pubblica:

- `afterfall_patch_1.0.0_to_1.0.5.zip`
- `afterfall_patch_1.0.1_to_1.0.5.zip`
- `afterfall_patch_1.0.2_to_1.0.5.zip`
- `afterfall_patch_1.0.3_to_1.0.5.zip`
- `afterfall_patch_1.0.4_to_1.0.5.zip`

Contenuto funzionale:

- Acque Biologo allineate a `10 minuti`
- durata client proto delle Acque allineata a `600s`
- `Pozione Verde(G)` `27112` allineata a `30 minuti`
- `Sigillo dell'Editto Regale` `70051` con hint `/remax`
- nessun cambio gameplay, server o C++

Le patch `1.0.0 -> 1.0.5` e `1.0.1 -> 1.0.5` includono:

- `Metin2_RelWithDebInfo.exe` pubblico gia' validato
- `pack/root.pck`
- `pack/locale.pck`
- `version.txt`

Le patch `1.0.2 -> 1.0.5` e `1.0.3 -> 1.0.5` aggiornano:

- `pack/root.pck`
- `pack/locale.pck`
- `version.txt`

La patch `1.0.4 -> 1.0.5` aggiorna solo:

- `pack/locale.pck`
- `version.txt`

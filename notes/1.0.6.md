# Afterfall Client 1.0.6

Patch client cumulativa focalizzata su selezione regno, mappe speciali e testi UI guerra.

Questa release pubblica:

- `afterfall_patch_1.0.0_to_1.0.6.zip`
- `afterfall_patch_1.0.1_to_1.0.6.zip`
- `afterfall_patch_1.0.2_to_1.0.6.zip`
- `afterfall_patch_1.0.3_to_1.0.6.zip`
- `afterfall_patch_1.0.4_to_1.0.6.zip`
- `afterfall_patch_1.0.5_to_1.0.6.zip`

Contenuto funzionale:

- `Chunjo` visibile ma non selezionabile nella scelta regno iniziale
- descrizione Chunjo aggiornata come regno `in ricostruzione`
- registrazione `AtlasInfo` per `metin2_map_t3` e `metin2_map_t4`
- registrazione `AtlasInfo` per `metin2_map_oxevent`
- fix path runtime OX per evitare il crash su `MapProperty.txt / Invalid Map Type`
- fix UI/minimap OX per evitare `KeyError: metin2_map_oxevent`
- testo reward guerra gilda/consorteria aggiornato da `pozioni` a `Yang`

Le patch `1.0.0 -> 1.0.6` e `1.0.1 -> 1.0.6` includono:

- `Metin2_RelWithDebInfo.exe` pubblico gia' validato
- `pack/root.pck`
- `pack/locale.pck`
- `pack/season1.pck`
- `version.txt`

Le patch `1.0.2 -> 1.0.6`, `1.0.3 -> 1.0.6`, `1.0.4 -> 1.0.6` e `1.0.5 -> 1.0.6` aggiornano:

- `pack/root.pck`
- `pack/locale.pck`
- `pack/season1.pck`
- `version.txt`

Non ci sono modifiche server, database, launcher o asset grafici inventati.
